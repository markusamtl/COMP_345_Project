Detailed Description

Represents an Advance order.

An Advance order moves armies from one territory to another, either reinforcing friendly territory or attacking an enemy.

void WarzoneOrder::Advance::execute ( )
overridevirtual

Execute the order. If valid, moves armies between territories. If target is enemy territory, resolves battle. If target is friendly, reinforces it.

Simultaneous battle: attackers have 60% chance kill a defender, defenders have a 70% chance to kill an attacker Attacker has to have 1 army on their source territory at a minimum
