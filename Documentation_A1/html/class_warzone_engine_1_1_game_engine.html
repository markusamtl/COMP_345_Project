Detailed Description

Central controller class managing the Warzone gameâ€™s state machine and flow.

The GameEngine class coordinates map loading, validation, player management, reinforcement assignment, order issuing, and order execution in a turn-based loop.

It also manages the transitions between different states, ensuring the proper sequence of commands according to the rules of Warzone.
