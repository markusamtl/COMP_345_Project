Detailed Description

Represents a Negotiate order.

A Negotiate order establishes temporary peace between two players, preventing them from attacking each other for the remainder of the turn.

This order is binding and does not require acceptance by the target player.

◆ execute()

void WarzoneOrder::Negotiate::execute ( )
overridevirtual

Execute the order.

If valid, this order modifies both players’ state by adding each player to the other’s list of neutral enemies (via Player::addNeutralEnemy()).

Later, when toAttack() is called, territories belonging to neutral enemies will not appear in the list of valid attack options, preventing attacks between the two players for the rest of the turn.
