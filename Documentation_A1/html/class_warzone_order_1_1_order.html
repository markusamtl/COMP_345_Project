Detailed Description

Abstract base class for all Warzone orders.

The Order class defines the interface that all specific Warzone orders must follow. Each subclass of Order (Deploy, Advance, Bomb, Blockade, Airlift, Negotiate) implements its own behavior for validation and execution according to the game rules.
