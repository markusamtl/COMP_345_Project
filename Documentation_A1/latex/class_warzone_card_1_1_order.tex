\doxysubsection{Detailed Description}
Abstract base class for all Warzone orders. 

The \doxylink{class_warzone_card_1_1_order}{Order} class defines the interface that all specific Warzone orders must follow. Each subclass of \doxylink{class_warzone_card_1_1_order}{Order} (\doxylink{class_warzone_order_1_1_deploy}{Deploy}, \doxylink{class_warzone_order_1_1_advance}{Advance}, \doxylink{class_warzone_order_1_1_bomb}{Bomb}, \doxylink{class_warzone_order_1_1_blockade}{Blockade}, \doxylink{class_warzone_order_1_1_airlift}{Airlift}, \doxylink{class_warzone_order_1_1_negotiate}{Negotiate}) implements its own behavior for validation and execution according to the game rules.
