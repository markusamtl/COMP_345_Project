\doxysubsection{Detailed Description}
Represents a \doxylink{class_warzone_engine_1_1_bomb}{Bomb} order. 

A \doxylink{class_warzone_engine_1_1_bomb}{Bomb} order removes half of the armies from an enemy territory. This order can only target territories not owned by the issuing player.
